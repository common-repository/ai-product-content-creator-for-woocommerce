== Description ==
Introducing the **AI Product Content Creator for WooCommerce**, a powerful plugin designed to boost your sales and conversion rates by generating SEO-optimized product titles and descriptions that will be more attractive to your users. Powered by the advanced AI of **ChatGPT**, this plugin allows you to effortlessly create compelling product descriptions tailored to your WooCommerce products.

Simply input your keywords, and the **AI product content creator** will generate unique, high-quality product titles and descriptions that enhance your store's SEO, making your products more discoverable. The plugin's smart algorithms ensure that every description is optimized for search engines based on the keywords you provide, helping your products rank higher and attract more customers.

Whether you're looking for an **AI WooCommerce product description writer** or a tool to streamline your content creation process, this plugin is the perfect solution for generating professional, keyword-optimized content with ease.

The AI Product Content Creator for WooCommerce plugin use AI to create compelling SEO optimized product titles, descriptions and short descriptions to help improve your sales and conversion rates.

Unlock the full potential of your WooCommerce store with this powerful **AI WooCommerce product description** tool!

== Frequently Asked Questions ==
= How do I use the plugin? =
You can generate product content by navigating to the product edit page.

= Where can I get support? =
Support is available through the plugin support page.

= Is the plugin free? =
Yes the plugin is free to use. But you must use your own API key. 

= Do I need an API Key? =
Yes you will need to use your own Open AI API. This is simple to create. You can follow the instruction on the [Open AI website](https://platform.openai.com/docs/quickstart). However the basic steps are as follows:

1. Log in or sign up to the OpenAI platform 
2. Go to the API Keys page 
3. Click Create new secret key
4. Name your API key 
5. Copy your key to your clipboard
6. Paste Key in the plugin settings

= Does the Plugin only work on WooCommerce Products? =
Yes, we focus on eCommerce and this plugin focuses on improving your products for better sales and conversions.

== Installation ==
1. Upload the `ai-product-content-creator-for-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure settings under 'AI Content Creator' in the admin dashboard.
